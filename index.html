Every track follows the same shape

Built so a beginner can walk in cold and walk out with a project.

Why it matters

Grounded in today

Each track opens by connecting the topic to something happening in the world right now — not abstract theory.

Learn by doing

One real workshop

Every track includes a hands-on activity: a pitch drill, a tabletop exercise, a teardown, a forecasting game.

Leave with direction

Skills & a next step

Participants leave with named skills, real career paths, and a project idea they can start the same week.

Who this is for

Built for a club, sized for a campus.

First-time attendees

No background assumed. Every session is written for someone hearing the topic for the first time.

Club organizers

Five ready-made tracks you can run as one summit, or split across a semester of smaller meetups.

Career explorers

Each track maps directly to real job families and a research idea worth pursuing afterward.

Start anywhere

Pick the track that pulls at you first.

There's no required order. Founders, historians, researchers, defenders, and analysts are all welcome in the same room.
